This is a simple chat application using Node JS and WebSocket!

First start by running:
    npm install
After the node_modules being generated

To create migrations in typeorm: 
    npm run migrate:create {name of the migration}

To run every migration:
    npm run migrate:up

To revert the last migration:
    npm run migrate:down

You'll need to execute:
    npm run dev
In order to execute the application
